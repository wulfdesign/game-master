# 🌌 GameMaster.io — Campaign Portal

The master landing gateway for **GameMaster.io**, a web-based toolkit for tabletop RPG game masters and players.

## 🎮 What Is This?

This is the public landing page for GameMaster.io. It provides:

*   🗺️ An overview of the platform and its features
*   🔗 Links to the **static demo site** for offline browsing at conventions and game nights
*   🔗 Links to the **live web application** (coming soon)

## 🏰 GeekFest West 2026

This site is being deployed for live tabletop RPG sessions at **GeekFest West** (July 2026). Players can scan QR codes at the table to browse creature cards, spell references, world lore, and campaign details — no internet required.

## 🔌 Offline Mode

The static demo site is designed to work with zero internet connectivity. All images, stylesheets, and scripts are bundled locally. Just open `index.html` in any browser.
